# KVKexporter
A simple script in development state, meaning more things will be added. The script lets the user export their Kovaak's settings fast and easy and it gives debug data about the KovaaK's game process without interfering with its memory. Enjoy!

This script is open-source so if there are any worries, you can compile the .ps1 script to an .exe from the source code .ps1 script in the main branch of this repo!
